Installation
------------

To use SparseDeepGP, first install it using pip:

.. code-block:: console

   $ git clone https://github.com/hchen19/sparse_dgp.git
   $ cd sparse_dgp
   $ pip install -e .
   # pip install -r requirements.txt # install requirements


.. .. code-block:: console

..    (.venv) $ pip install sparse_dgp
